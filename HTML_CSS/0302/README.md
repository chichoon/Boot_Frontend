## HTML Tag
- 브라우저에게 이게 뭔지 설명해주기 위한 장치
- 야 이게 타이틀이고 이게 링크야.. 하는 것
- html에는 이미 정의되어 있는 tag들이 있으므로, 그것들을 사용하면 브라우저가 그에 따른 동작을 취한다
    - 형식 : <태그>내용물</태그> (닫아줘야함!)
    - 정의되지 않은 뜬금없는 태그를 써도 되지만, 브라우저는 이걸 이해하지 못할 것
    - 브라우저는 해당 태그를 이해하지 못해도 억지로 실행은 시켜준다 (에러가 났는지 안알려줌)
